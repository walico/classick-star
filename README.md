# Classic Star Home Maintenance LLC

Website for Classic Star Home Maintenance LLC — **Fix it right, Fix it once.**

## File structure

```
classic-star/
├── index.html          # Home: services, hero, CTA
├── about.html          # About: Aim, Mission, Vision
├── css/
│   └── custom.css      # Clouds, scroll-reveal, custom animations
├── js/
│   └── main.js         # Scroll-triggered reveal logic
├── assets/             # Add your images here (optional)
└── README.md
```

## Run the site

- Open **index.html** in your browser, or
- With XAMPP: **http://localhost/noninestudio/classic-star/**

No build step needed — the site uses the Tailwind CSS CDN.
